Why We Built It

The Challenge of
Large Conferences

Managing where hundreds of pastors and dignitaries sit across multiple sessions is complex, error-prone, and stressful without the right tools.

📋
Spreadsheet Chaos
Paper lists and spreadsheets get lost, go out of date, and can't show real-time arrival status.
🪑
No Visual Overview
Without a floor plan view, protocol officers can't quickly see which seats are taken or empty.
👥
Team Coordination
Multiple protocol officers working from different lists leads to double-bookings and missed guests.
🔄
Repeated Setup
Each conference session means starting over — re-entering the same pastors, re-assigning seats.
The Solution

One System, Every Event

Pastors' Protocol Central brings every part of dignitary management into a single, beautiful web app your whole protocol team can access together.

🏟️
Set Up Your Conference
Create the event, pick the auditorium, and add sessions for each service.
›
👤
Build the Roster
Add dignitaries to a shared directory and assign them to specific sessions.
›
🗺️
Assign Seats
Click seats on an interactive floor plan to place each pastor exactly where they belong.
›
📡
Track Live
Mark arrivals, seating, and absences in real time from the control centre.

Quick Start

Up & Running in Minutes

Getting started takes four simple steps. Once your first conference is set up, future events take even less time.

1
🔑
Sign In
Open the app in your browser and sign in. New users register in seconds.
2
🏟️
Create a Conference
Name your event, set the date, pick the auditorium, and add sessions.
3
👤
Add Dignitaries
Build or reuse the dignitary directory, then assign them to each session.
4
🗺️
Seat & Track
Use the floor plan to assign seats, then go live on event day.
💡 Pro tip: Use Import Arrangement to copy an existing session's seating plan into a new session — saving hours of re-entry.
